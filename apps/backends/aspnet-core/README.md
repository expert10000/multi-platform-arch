# DZONE ASP.NET Core Backend

ASP.NET Core implementation of the shared platform API contract.

```powershell
dotnet run --project apps/backends/aspnet-core/DzoneAspNetCoreBackend.csproj --urls http://127.0.0.1:3300
```

It serves the same central admin, web host, runtime admin pages, workspace/document routes, file upload route, and job routes as the Node, Spring Boot, and Python backends.
